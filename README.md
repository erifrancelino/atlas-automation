# Atlas Automation

## Demonstração

Vídeo da automação funcionando:

[(https://youtu.be/STSScXno3VY)]


Automação desenvolvida em Python para migração de visitas educacionais para o sistema Atlas Concierge.



## Objetivo


O projeto automatiza o cadastro de visitas a partir de uma planilha Excel, eliminando o preenchimento manual de centenas de registros.



## Tecnologias utilizadas



\* Python

\* Selenium

\* Pandas

\* WebDriver Manager



## Funcionalidades



\* Login automático no sistema Atlas

\* Leitura de dados a partir de planilha Excel

\* Navegação automática pelo calendário

\* Seleção de categorias

\* Cadastro de grupos e turmas

\* Inserção de educadores

\* Registro de participantes

\* Inclusão de observações

\* Salvamento automático das visitas



## Resultado



A automação reduziu significativamente o tempo necessário para migrar registros históricos para o sistema Atlas, evitando erros manuais e aumentando a produtividade do processo.



## Observação



Por questões de segurança, credenciais de acesso e dados institucionais não estão incluídos neste repositório.
